Python API Reference

Complete reference for using AI-IQ as a Python library. All methods are thread-safe via SQLite WAL mode.

Installation

# Basic installation
pip install ai-iq

# With semantic search (vector embeddings)
pip install ai-iq[full]

Memory Class

The Memory class provides the main interface to AI-IQ's persistent memory system.

Memory(db_path=None)

Initialize memory system with optional custom database path.

Parameters:
  • db_path (str, optional) — Path to SQLite database. Defaults to ./memories.db
from ai_iq import Memory

# Default database (./memories.db)
memory = Memory()

# Custom database path
memory = Memory("my_memories.db")

add(content, category="general", tags=None, project=None, priority=0, related_to=None, expires_at=None)

Add a new memory to the system.

Parameters:
  • content (str, required) — The memory content
  • category (str) — Category: project, decision, preference, error, learning, pending, architecture, workflow, contact, general
  • tags (str or list) — Comma-separated tags or list of tags
  • project (str) — Project name to associate this memory with
  • priority (int) — Priority level 0-10 (higher = more important)
  • related_to (int) — ID of related memory to link to
  • expires_at (str) — Expiry date in YYYY-MM-DD format
Returns: int — Memory ID if created, None if duplicate 
# Simple add
memory.add("Docker containers need network_mode: host")

# With full options
mem_id = memory.add(
    "Always use environment variables for secrets",
    category="decision",
    tags=["security", "best-practice"],
    project="MyApp",
    priority=9
)

search(query, mode="hybrid", include_token_estimate=False, compact=True)

Search memories with hybrid keyword + semantic search.

Parameters:
  • query (str) — Search query string
  • mode (str) — Search mode: "hybrid" (default), "keyword", or "semantic"
  • include_token_estimate (bool) — Add token_estimate field to results
  • compact (bool) — Return minimal fields (id, content truncated, category, tags). Default: True
Returns: List[Dict] — List of memory dictionaries 
# Compact search (default) - minimal data transfer
results = memory.search("redis caching")
for r in results:
    print(f"#{r['id']}: {r['content']}")  # Content truncated to 100 chars

# Full search - get all data
results = memory.search("redis", compact=False)
for r in results:
    print(r['content'])  # Full content

# With token estimates for budget planning
results = memory.search("redis", include_token_estimate=True)
total_tokens = sum(r['token_estimate'] for r in results)
print(f"Reading all results would cost ~{total_tokens} tokens")

get(mem_id)

Get full details for a single memory by ID.

Parameters:
  • mem_id (int) — Memory ID
Returns: Dict or None — Dictionary with all memory fields, or None if not found 
mem = memory.get(42)
if mem:
    print(mem["content"])
    print(mem["category"])
    print(mem["created_at"])

update(mem_id, content)

Update a memory's content. Auto-updates tags, revision count, and re-generates embeddings.

Parameters:
  • mem_id (int) — Memory ID to update
  • content (str) — New content
memory.update(42, "Updated content here")

delete(mem_id)

Soft-delete a memory (marks as inactive, recoverable).

Parameters:
  • mem_id (int) — Memory ID to delete
memory.delete(42)

list(category=None, project=None, tag=None, stale_only=False, expired_only=False)

List memories with optional filters.

Parameters:
  • category (str) — Filter by category
  • project (str) — Filter by project name
  • tag (str) — Filter by tag (partial match)
  • stale_only (bool) — Only show stale memories
  • expired_only (bool) — Only show expired memories
Returns: List[Dict] — List of memory dictionaries 
# Get all memories for a project
mems = memory.list(project="MyApp")

# Get all pending items
todos = memory.list(category="pending")

# Get stale memories that need review
stale = memory.list(stale_only=True)

stats()

Get database statistics.

Returns: Dict — Dictionary with memory counts, vector counts, etc. 
stats = memory.stats()
print(f"Total memories: {stats['total']}")
print(f"Active memories: {stats['active']}")
print(f"Categories: {stats['categories']}")
print(f"Vector embeddings: {stats['vectors']}")

Search Modes

AI-IQ supports three search modes, combined via Reciprocal Rank Fusion (RRF).

Hybrid Search (default)

Combines keyword search (FTS5), semantic search (vector similarity), and graph traversal. Best for most use cases.

results = memory.search("redis caching")

Keyword Search

Fast exact/partial text matching using SQLite FTS5. Best for specific terms or technical names.

results = memory.search("redis", mode="keyword")

Semantic Search

Conceptual similarity using vector embeddings (384-dim all-MiniLM-L6-v2). Best for finding related concepts.

results = memory.search("caching strategy", mode="semantic")

Complete Example

from ai_iq import Memory

# Initialize
memory = Memory("my_app.db")

# Add memories
id1 = memory.add(
    "Docker containers need network_mode: host for Redis",
    category="learning",
    tags=["docker", "redis", "networking"],
    project="Backend"
)

id2 = memory.add(
    "Always use environment variables for secrets",
    category="decision",
    tags=["security", "best-practice"],
    priority=9
)

# Search
results = memory.search("docker redis")
print(f"Found {len(results)} results:")
for r in results:
    print(f"  #{r['id']}: {r['content'][:60]}...")

# Get specific memory
mem = memory.get(id1)
print(f"\nMemory #{mem['id']}:")
print(f"  Content: {mem['content']}")
print(f"  Tags: {mem['tags']}")
print(f"  Created: {mem['created_at']}")

# Update
memory.update(id1, "Docker ALWAYS needs network_mode: host for Redis")

# List by project
backend_mems = memory.list(project="Backend")
print(f"\n{len(backend_mems)} memories in Backend project")

# Statistics
stats = memory.stats()
print(f"\nTotal: {stats['total']}, Active: {stats['active']}")
print(f"Categories: {list(stats['categories'].keys())}")

# Clean up
memory.delete(id2)
